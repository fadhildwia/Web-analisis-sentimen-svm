KLASIFIKASI ANALISIS SENTIMEN PENGGUNA TWITTER TERHADAP LAYANAN INTERNET BIZNET MENGGUNAKAN ALGORITMA SUPPORT VECTOR MACHINE (SVM)

Harap Di Baca Terlebih Dahulu

1. Siapkan dataset sentimen dari twitter dengan format .csv yang terdiri dari dua kolom. kolom pertama untuk sentimen berikan nama "Text" dan kedua untuk sentimen berikan nama "Labels".

2. Pilih menu Upload Data untuk import data sentimen yang akan digunakan atau untuk mengganti dataset baru.

3. Lakukan text preprocessing pada menu Preprocessing.

4. Lakukan pembobotan fitur dengan memilih menu TD-IDF.

5. Lakukan pengujian analisis sentimen dengan algoritma SVM pada menu Klasifiasi SVM.

6. Hasil dari pengujian akan tampil pada menu Klasifikasi SVM terdiri dari akurasi, precision dan recall. Cofusion matrix akan tampil setelah nilai akurasi.

7. lakukan pengujian pada model yang telah dibuat dengan memasukkan kalimat baru yang tidak ada di dataset dan lihat hasilnya apakah analisis sentimen dengan klasifikasi SVM berjalan dengan baik.
